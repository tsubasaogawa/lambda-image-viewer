The project utilizes the following technologies:

*   **Backend:** AWS Lambda, CloudFront, DynamoDB, S3.
*   **Languages:** Go (for Lambda functions), JavaScript (for some Terraform assets).
*   **Frameworks/Tools:** Serverless Framework v3.34, Terraform v1.12+, Go 1.21.
*   **Deployment Version Management:** `slsenv`, `tfenv`, `goenv` are recommended for managing tool versions.
