# データモデル

この機能は主にファイル生成を行い、永続化するデータモデルは存在しません。

## 主要エンティティ

- **設定ファイル (`.gemini/config.yaml`, `.gemini/commands/*.toml`)**
  - **説明**: spec-kitの動作を定義するためのファイル群。
  - **属性**:
    - `config.yaml`: プロジェクト全体の設定。
    - `commands/*.toml`: 各spec-kitコマンドの動作定義。
  - **関連**: 他のエンティティとの直接的な関連はない。

- **プロジェクト**
  - **説明**: spec-kitが導入される対象のコードベース。
  - **属性**:
    - プロジェクトルートパス
    - 既存のファイル構造
  - **関連**: 設定ファイルはプロジェクト内に配置される。
