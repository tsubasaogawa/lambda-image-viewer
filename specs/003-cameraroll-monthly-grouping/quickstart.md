# Quickstart: カメラ画像の月別グルーピング

**Feature Branch**: `003-cameraroll-monthly-grouping`
**Created**: 2025-11-14

## 1. 概要

このドキュメントは、カメラロールの月別グルーピング機能のフロントエンド実装をローカル環境で素早く確認するための手順を提供します。

## 2. 前提条件

- 開発環境がセットアップされており、プロジェクトの依存関係が解決されていること。
- `003-cameraroll-monthly-grouping` ブランチがチェックアウトされていること。

## 3. 変更の確認手順

1.  **ローカルサーバーの起動**:
    プロジェクトのルートディレクトリで、アプリケーションを起動します。具体的なコマンドはプロジェクトのセットアップによって異なりますが、通常は以下のいずれかです。
    ```bash
    # 例: Goアプリケーションの場合
    go run src/viewer/cmd/viewer/main.go
    # または、Serverless Frameworkを使用している場合
    npm start # または serverless offline
    ```
    （注: 正確な起動コマンドは、プロジェクトの `README.md` や `package.json` を参照してください。）

2.  **ブラウザでのアクセス**:
    Webブラウザを開き、カメラロールのURLにアクセスします。通常は `http://localhost:8080/cameraroll/` のようなURLです。

3.  **機能の確認**:
    -   ページが読み込まれた後、画像が月ごとにグループ化されて表示されていることを確認します。
    -   各月のグループが、`<h2>YYYY-MM</h2>` のような見出しと、`<ul id="thumbs-YYYY-MM">` の形式のIDを持つ `<ul>` タグで囲まれていることを確認します。
    -   ブラウザの開発者ツール（F12キーなどで開く）を使用して、DOM構造が期待通りに更新されていることを確認します。
    -   ページの読み込み時に、目立った画面のちらつきやJavaScriptエラーが発生しないことを確認します。

## 4. クリーンアップ

ローカルサーバーを停止します。
